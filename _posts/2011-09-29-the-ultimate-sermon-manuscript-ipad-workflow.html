Introduction

Why a sermon manuscript


When I preach, I prefer to have a sermon manuscript. Now this does not mean that I read a manuscript. I am a firm believer that there is a difference between reading and preaching, but I don’t feel prepared unless I have a manuscript written, and I have wrestled with ways to get that manuscript into the pulpit in a manageable way.

My iPad fills that need for a couple of reasons. First, my average manuscript is about 15 typewritten pages long, and with my iPad, I don’t have to cary 15 pages into the pulpit and constantly be shuffling paper around. Second, I can use the table of contents feature in iBooks to look at a rough outline of my sermon while I preach, and touch the points and sub points if I need to refer to the details of my manuscript.

This has been by no means an easy thing to figure out, and I am constantly tweaking it to make it better, but I think I’ve finally stumbled upon a way to do this manageably.

What you’ll need


All you really need to do this is

  • an iPad (you could also use any other tablet or ereader, but these instructions are for the iPad)
  • a plain text editor (I use nvAlt on the mac which syncs with my iPad so I can always write.)
  • Calibre - a free tool that converts plain text to ePub.
  • A program to make a cover image - I use InDesign and have a template set up, then convert it to a jpg file. This is totally optional, your sermon will work just fine without a cover image.

Write the sermon in plain text using Markdown


I used to write everything in either Evernote or Microsoft Word. Both of these programs have a lot of great features, but they also have a lot of distracting features. I have found that (for me) the best way to write is in a simple plain text file using a syntax called Markdown.

What is Markdown? Markdown is a way to write HTML without writing HTML. You get the structure of HTML (Heading 1, Heading 2, Emphasis, Strong, Blockquote, Etc.) Without having to worry about tags. This took me literally ten minutes to learn and it saves so much time. It just lets me write, without worrying about formatting, without worrying about what programs can and can’t open my files. I can’t recommend it enough - it is super simple.

I write my sermons in a free mac app called nvAlt, a fork off of another free mac app called Notational Velocity. The benefits of this app for me is that it has a full screen editor, which makes all of the distractions go away, and it automatically saves and syncs online with SimpleNote, which means I can access the text files on my iPad whenever I have ideas.

I’m not going to tell you how to write the sermon, thats the real fun part, just how to set up an already written sermon so you can view it on your iPad.

Set up Calibre for your iPad


Calibre is a great program with an awful interface. I usually don’t use software that is this ugly, but it does the job, costs me nothing, and I can’t find an alternative so I am stuck with it.

Setting up Calibre is a little overwhelming at first, but if you do it write you should only have to do it once.

Setting up Calibre preferences for this workflow


First open the Calibre preferences pain by pressing “Command + ;” on a mac or going to Edit > Preferences on Windows.

Set up Input Options

  • To set up the input options, click the “Input Options” button. 
  • Then click the “plain text” button from the column on the left.
  • Under “formatting style”, select “Markdown” 
  • Make sure that the “Do Not Insert Table of Contents …” box is unchecked
  • Click the apply button on the top bar.

Set up the Common Options

  • To set up the common options, click the “Common Options” button. 
  • Then click on the “Page Setup” button in the column on the left. 
  • Set the Output profile as “iPad” and the input profile as “Default Input Profile” 
  • Click on the “Table of Contents” button in the column on the left. 
  • In the three boxes type
    • In the “Level 1 TOC” box type “//h:h1”
    • In the “Level 2 TOC” box type “//h:h2”
    • In the “Level 3 TOC” box type “//h:h3”


  • Click the apply button on the top bar.

Set up the Output Options

  • To set up the output options, click the “Output Options” button. 
  • Select “ePub Output” from the column on the left.
  • Check “No Default Cover” 
  • Click the apply button on the top bar.

Never look at that preference pane again!

Drag the plain text file into Calibre


Now that Calibre is set up, it is a fairly easy task to turn our Markdown formatted plain text into pretty iPad eBooks with usable tables of contents. We just need to add the file to Calibre.

Adding the file to Calibre is as easy as …

  • opening the Calibre window
  • finding your plain text file in finder or windows explorer
  • dragging the file onto the Calibre window.

Convert the plain text file into ePub

  • To convert your file into ePub format, just make sure that your file is selected in the main window and click on the “Convert Books” button. (It looks like a book.) 
  • In the window that pops up you only need to change things on the first section.  
    • Change the “Output Format” to ePub.
    • Change the Title, Author, Etc, information if you want.
    • Drag an 600 x 800 pixel cover image into the cover panel if you want a cover.
    • Click “OK”



Get the ePub file into iTunes


Getting the ePub file into iTunes is fairly easy. You just need to get to the ePub file on your computer and drag the file to the iTunes icon in the dock (if you are using a mac) or to the iTunes window (on windows). The easiest way to get to the ePub file is to select your newly created manuscript in the main Calibre window and on the right, underneath the large book icon, click the text that says “Click to Open.” You can then just drag the file onto either the iTunes icon or the iTunes window.

Sync up your iPad


Now all you have to do is sync up your iPad. Plug it in, wait a few minutes and you’ll have a very functional sermon manuscript.

Conclusion


While this may seem like a lot of work, it goes by pretty quickly after you’ve done it one or two times and the benefits are well worth it.  Below is a short video of the finished product.  (Sorry, it's not very good.)

Let me conclude with a couple of tips about preaching from an iPad.

  • Turn the brightness way down. If the brightness is all the way up it is going to look like your are preaching with a spotlight on your face. You don't necessarily want to hide the fact that you are using an iPad, but you don't want it to be a distration.
  • If you can, preach without glasses. Any backlit screen is going to glare in your glasses and people will notice.
  • Lock the screen orientation. The easiest way to do this is to set it up in system preferences so the little switch on the side (above the volume button) locks the orientation. This will keep you from fussing over screen orientation.
  • Consider using a non-obvious iPad cover. I have an iPad cover that my wife made for me out of an old day planner. It just looks like a leather book, I don't use it all the time, but it does make it a little less obvious that I am preaching from an iPad.
