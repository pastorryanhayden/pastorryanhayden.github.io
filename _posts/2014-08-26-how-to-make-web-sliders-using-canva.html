I've been doing a lot of web design work lately (and thus, not a lot of blogging.)  As a part of a project I'm working on, I've created a long manual on how to use a free online program called Canva to make slider images for your home page.

Slider images are one of the trickiest things to teach clients how to make because they invariably require using some kind of graphic design software.  You won't find anything easier than Canva.

So I'm going to share that file via my blog for two reasons:

  1. I want pastors to know how to use Canva to do some basic graphic design projects.
  2. I want prospective web design clients to see the kind of training material I include with my web work.


Check out the link to the google doc below (Warning, some of it, like the dimensions, is web site specific.)
